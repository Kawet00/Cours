# Exemple-HTML

Un petit exemple simple pour apprendre et comprendre les bases du HTML.

Description
-----------
Ce dépôt contient un exemple minimal d'une page HTML (`index.html`) destiné aux débutants. Le but est de montrer comment le code HTML se traduit visuellement dans le navigateur et d'expliquer simplement comment ouvrir et tester la page.

Comment l'utiliser
------------------
- Ouvrir directement : double-cliquez sur le fichier `index.html` dans votre explorateur de fichiers — il s'ouvrira dans votre navigateur par défaut.

Structure du dépôt
------------------
- `index.html` : la page HTML d'exemple à consulter.
- `README.md` : ce fichier, expliquant l'objectif et l'utilisation du dépôt.

Bonnes pratiques
-----------------
- Utilisez un éditeur de texte (VS Code, Sublime, Atom...) pour ouvrir et modifier `index.html`.
- Rafraîchissez le navigateur pour voir les modifications après sauvegarde.
