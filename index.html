Mini-cours Web

Présentation du site

Ce site rassemble des mini-cours. L’objectif est de revoir rapidement les bases indispensables pour :

  • structurer une page avec HTML,
  • mettre en forme cette page avec CSS,
  • manipuler les données avec SQL (MySQL / phpMyAdmin).

Choisissez un mini-cours ci‑dessous pour commencer. Vous pouvez revenir sur cette page d’accueil à tout moment avec le raccourci du navigateur (retour) ou en tapant l’URL du dossier racine.

Conseils d’utilisation en classe

  • Ouvrez les mini-cours dans un navigateur, puis testez le code dans un éditeur.
  • Pour SQL, utilisez WAMP et phpMyAdmin pour reproduire les requêtes.
  • Modifiez les exemples pour voir immédiatement l’impact de vos changements.
